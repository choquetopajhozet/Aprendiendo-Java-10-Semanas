# Semana 1: Sistema de Calificaciones

 Programa que gestiona las notas de 5 estudiantes.

## Funcionalidades

 - Almacena 5 estudiantes con 3 notas cada uno
 - Calcula promedios individuales y promedio general
 - Determina estudiantes aprobados y reprobados
 - Muestra la nota máxima y mínima del curso

## Cómo ejecutar

 1. Entrar a la carpeta

 cd semana-01-calificaciones

 2. Compilar

 javac Main.java

 3. Ejecutar

 java Main
